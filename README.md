# Installation

### Limitations
This script was made for Endeavour OS, so it may not work anywhere else.

### Steps
- clone the repo through SSH
- run the script using `cd dot-files && bash ./install.sh`
- set default terminal
- reboot

### Followup
Set your git identity using `git config --global user.email "you@example.com"` and `git config --global user.name "Your Name"`
If you want to find the ssh key for github, run `cat ~/.ssh/id_ed25519.pub`
